How to Run:
This program is tested for localhost i.e "127.0.0.1".

To Run Client:
./client 127.0.0.1 <same as client port number>

To Run Server:
./server <Any Port number>

How to Compile:
1.Copy all the Source code in a Linux PC is Particular folder.
2.mohan.bmp is added as file to be transfer.
3.The Server will change the file name as 123mohan.bmp since its in same folder and same machine.
4.To compile both server and client run:
	make all
only server
	make server
only client
	make client
cleanall
	make clean

The Server file is saved as mohan.bmp;
